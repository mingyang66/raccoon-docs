###### 一、macos设置环境变量

```sh
export 'key'
```

###### 一、macos查看export命令设置的环境变量

- 方案1：

```sh
echo $PATH
```

- 方案2：

```sh
printenv
```

- 方案3：

```sh
env
```

- 方案4：

```sh
vi ~/.bash_profile
```

###### 三、快捷键
