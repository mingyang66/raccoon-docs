###### 破窗理论

> 一个破碎的窗户，如果长期以来没有修好，就会给建筑中的居民逐渐带来一种被废弃的感觉——感觉管理者并没有在意这栋建筑。这样会再有一个窗户被损坏。人们也开始乱丢东西、涂鸦，建筑被严重损坏。很快这栋建筑的损坏程度使得其主人都不想再维修它了，放弃的意愿就成了现实。
