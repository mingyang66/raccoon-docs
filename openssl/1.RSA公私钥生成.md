##### 一、生成私钥（Private Key）

```
openssl genpkey -algorithm RSA -out private_key.pem -pkeyopt rsa_keygen_bits:2048
```

- genpkey：较新的通用密钥生成命令（推荐）。

- -algorithm RSA：指定算法为RSA。

- -out private_key.pem：生成的私钥保存为private_key.pem文件。

- -pkeyopt rsa_keygen_bits:2048：制定秘钥长度为2048位（安全标准）。如果需要更高安全标准可以更改为4096.

##### 二、从私钥中提取公钥（Public Key）

```
openssl rsa -pubout -in private_key.pem -out public_key.pem
```

- -pubout：表示输出公钥。

- -in private_key.pem：输入刚才生成的私钥文件。

- -out public_key.pem：生成的公钥保存为 public_key.pem。

##### 三、验证生成结果

1. 查看私钥

```
openssl rsa -in private_key.pem -text -noout
```

输出应包含 Private-Key, modulus, publicExponent 等信息。

2.查看公钥信息

```
openssl rsa -pubin -in public_key.pem -text -noout
```

输出应包含 Public-Key, modulus, publicExponent 等信息。

3.默认生成的 private_key.pem 通常是 PKCS#8 格式（以 -----BEGIN PRIVATE KEY----- 开头），这是现代标准，兼容 Java, Go, Node.js 等。
